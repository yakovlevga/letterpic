                        JS:
                        $(".letterpic").letterpic({
                            fill: 'color',
                            initial: {
                                color: {
                                    "A" : "red",
                                    "B" : "blue",
                                    "M" : "green",
                                }
                            }
                        });

                        HTML:
                        
                        <div class="letterpic" title="Mike Green" data-letterpic-key="M"></div>
Now you can define attribute data-letterpic-key to manualy set color dictionary value of user.
By default this key is equals to users initials from title attribute.
Also you can pass maxInitialsLength as parameter of $.letterpic, to define max length of initals string on avatar,
by default it's value is 2.
